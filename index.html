Patchwork Toolkit

A toolkit for connecting various devices (i.e. Cameras, Meters, Arduinos, RaspberryPI's, Plugwise, etc) into a network of things or, in a more broad case - Internet of Things (IoT).
A lightweight set of components for quick composition of an infrastructure for smart home or office environment.

Lightweight

No heavy Java OSGI or similar containers. No numerious Python eggs/wheels dependencies to drag around. No lengthy kilometers of complex XML declarations to edit. Each core component is written in Go and distributed as a single executable, simple JSON configuration.


More Info

Crossplatform

The core components can be compiled for every platform supported by Go language. We have tested the toolkit on Linux, OSX, Windows 7 and a special attention has been paid to RaspberryPI and BeagleBone Black pocket-size computers.


More Info

Relies on Standards

A harmonized API for diverse devices that relies on widely accepted standards (i.e. RESTful/SOAP/CoAP/MQTT).

Getting Started

You should be able to use the toolkit on any platform that is supported by Go programming language.

1

Download the binary distribution for your platform and unzip it.
sdf

2

Use provided samples to configure Patchwork components and sample devices.

3

Start the components using provided Procfile or manually and explore the services and devices in the catalogs.
